# Project Plan - Spell Checker

## Goal

Build a simple and clear spell checker that matches the CYF rubric.

## Milestones

- [x] Setup project files and folder structure
- [x] Add dictionary data (`words.json`)
- [x] Build spell check logic
- [x] Build UI with clear messages
- [x] Add "Add to dictionary" action
- [x] Handle punctuation and hyphen words
- [x] Ignore words with capital first letter
- [x] Add unit tests
- [x] Add GitHub workflow for auto deploy
- [x] Add issue and PR templates

## Definition of Done

- Repo link is ready
- Deployed site link is ready
- Tests run and pass
- Rubric checklist is complete
- Code can be explained line by line
